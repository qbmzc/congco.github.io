## 配置文件

```yml
spring:
  redis:
    database: 0
    host: 127.0.0.1
    pool:
      max-active: 8
      max-idle: 8
      max-wait: -1
    port: 6379
    timeout: 0
```

## 错误信息

```bash
Error redis clients jedis HostAndPort - cant resolve localhost address
```

## 解决方案

查看Linux系统的主机名

```bash
hostname

cong
```

修改`/etc/hosts`

```bash
sudo vim /etc/hosts
```

添加`127.0.0.1 hostname`

```bash
# Host addresses
127.0.0.1  localhost
127.0.1.1  cong
```
