## 查看交换分区

```bash
free -h
```

|       | total | used | free | shared | buff/cache | available |
| ----- | ----- | ---- | ---- | ------ | ---------- | --------- |
| Mem:  | 62G   | 58G  | 507M | 2.7M   | 3.4G       | 3.2G      |
| Swap: | 4.0G  | 14M  | 4.0G |        |            |           |

## 命令

```bash
swapoff -a
swapon -a
# 另外保证重启生效需要 编辑`/etc/fstab`，注释掉`swap`那一行
```
