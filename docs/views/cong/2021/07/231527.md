## less

```bash
less -N log.log
# /* 搜索内容 n/N 下一个/上一个

```

## grep

```bash
# -c 行数
grep -c "content" log.log

# -E 正则
grep -E "[1-9]+" log.log

# 关键词所在行
grep -i "content" log.log
```

## cat

```bash
cat log.log|head -n 5
cat log.log|tail -n 5
cat log.log|grep "content"
```

## tail

```bash
tail -f log.log # 动态查看日志
tail -n 5 log.log
head -n 5 log
tail -n +5 log # 5行以后的
```

## sed

```bash
sed -n "5,10p" log # 5-10行
```

## find 

```bash
find .|xargs grep -ri "content" # 所有文件中是否含有某个字符串
```
