## 字体安装

```bash
yarn sarasa # 安装更纱黑体
```

## 更新缓存

```bash
fc-cache -fv  
```

## 硬盘测速

```bash
yay hdparm #安装测速工具
df -h # 查看硬盘信息
sudo hdparm -t /dev/sdc1 #开始测试
# 结果
/dev/sdc1:
 Timing buffered disk reads: 324 MB in  3.02 seconds = 107.43 MB/sec
```
