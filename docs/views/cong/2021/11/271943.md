## jps:虚拟机进程状况工具

jps(JVM Process Status Tool):可以列出正在运行的虚拟机进程，并显示虚拟机执行主类（Main.class）名称以及这些进程的本地虚拟机唯一ID

jsp命令格式

```bash
jps [options] [hostid]
# 例如

jsp -l

1928 
46284 jdk.jcmd/sun.tools.jps.Jps9009
```

- 其他常见选项

|选项|作用|
|---|---|
|-q|只输出绿米的，省略主类的名称|
|-m|输出虚拟机进程启动时传递给主类main()函数的参数|
|-l|输出主类的名称，如果执行的JAR包，则输出JAR路径|
|-v|输出虚拟机进程启动时的JVM参数|
