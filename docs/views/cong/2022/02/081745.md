##                                  写入文件
```python
# 打开
f = open('test.txt', 'w')
# 写入
f.write('hello world')
# 关闭
f.close()
```

- `w`代表写入
- `r+`代表读写
- `a`代表追加



### with语法写入
```python
with open('test.txt', 'a')as f:
    f.write('hahahaha')
```
## 读取数据


```python
# 打开
f = open('test.txt', 'r')
# 读取
for line in f:
    print(line)
# 关闭
f.close()
```
### with语法读取
```python
with open('test.txt', 'r')as f:
    for line in f:
        print(line)
```
