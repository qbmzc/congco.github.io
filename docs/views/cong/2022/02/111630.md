## 问题
sudo source找不到命令
问题是source是一个bash build-in命令(不是程序 – 如ls或grep)。

## 以root用户身份登录，然后执行source命令

```shell
sudo -s #切换到root账户
source /etc/profile # 生效环境变量
```

- 另一种解决方案

以root用户启动一个新的bash

```shell
sudo bash
```
