## 问题

文件上传之后发现中文文件名称出现？？乱码。

## 解决办法

上传文件如果通过zuul的话, 在接口地址前面加上zuul, 如果你是file/upload, 那么就要在页面使用时, zuul/file/upload

- 方案2

add this config in zuul service application.properties

```properties
zuul.servlet-path=/
```
