## 问题

```bash
FATAL ERROR: CALL_AND_RETRY_LAST Allocation failed - JavaScript heap out of memory error.
```

## 解决办法

```shell
## 设置环境变量
export NODE_OPTIONS="--max-old-space-size=4096"
```
