## 问题

docker commit :从容器创建一个新的镜像。

## OPTIONS说明：

-a :提交的镜像作者；

-c :使用Dockerfile指令来创建镜像；

-m :提交时的说明文字；

-p :在commit时，将容器暂停。

## 实例

```bash
docker commit -a "snownull" -m "my-app" aaaaa  my-app:v1.0

# 登陆 docker hub
docker login 

# 打标签
docker tag [old] [new]
# 推送本地镜像到仓库
docker push [images:tag]

```
