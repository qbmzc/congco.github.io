## 安装依赖

```bash
pip install  fastapi uvicorn
```

## 实例

```python
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
async def root():
    return {"message": "Hello World"}


@app.get("/hello/{name}")
async def say_hello(name: str):
    return {"message": f"Hello {name}"}

```

## 启动

```bash
uvicorn main:app --reload 

# http://127.0.0.1:8000/
# http://127.0.0.1:8000/docs 文档
```
