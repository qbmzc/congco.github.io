## 代码实例

```python
import requests

url  = "http://0.0.0.0:8000/%E5%A6%82%E6%A2%A6.wma"
payload = {}
headers = {
    'appKey': '12345678'
}

response = requests.request("GET", url, headers=headers,data=payload)

print(response.status_code)

with open("/home/cong/Downloads/a.wma", "wb") as f:
    f.write(response.content)

```
