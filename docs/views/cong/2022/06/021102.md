## 需求

需要在日志文件中查找报错的请求接口信息.

## 多条件筛选

```bash
# 关键词后七行,其中包含的错误信息为'The operation is not valid',且包含key2
grep -a7 'keyword'  *.log | grep -n6 "The operation is not valid" |grep "key2"> 159_1_error.txt 
```

## 查找关键信息

```bash
# 输出统计时间的日志信息
grep 'total time' *.log > r2.txt
```
