## 问题

```bash
minikube start
# 该命令默认连接的仓库是 Google container register,在国内无法访问
```

## 解决方案

1. 代理
2. 设置使用国内仓库


```bash
# 删除之前的启动配置
minikube delete --all
# 指定国内镜像
minikube start --image-repoisitory='cn'
```
