## 问题

OS X的预览可以显示元数据，尽管子集的数量有限（在“工具”>“显示检查器”下）。但是，使用“预览”看不到嵌入的字体信息。

## 命令行查看pdf字体

```shell
# OS x/Linux
strings /path/pdf.pdf |grep -i FontName
# Win
findstr FontName C:\\Path\\pdf.pdf
```

## 工具查看pdf字体


- pdffonts
- poppler
- xpdf
