## python3调用shell命令常用方法

- os.system(cmd)

```python
import os

os.system('ls -la')
```

- os.popen(cmd,mod)

```python
import os
var = os.popen('ls -la').read()
print(var)
```

- subprocess

```python
import subprocess 

subprocess.call(command,shell=True)

s = subprocess.run(['python3','-m','pip','list'])
print(s)

p = subprocess.Popen(["java", "-version"])
p.wait()
print(p.returncode)
```
