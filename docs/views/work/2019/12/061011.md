## 代码

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script type="text/javascript">
      function speak() {
          var utterThis = new SpeechSynthesisUtterance();
          utterThis.text="床前明月光,疑是地上霜。举头望明月，低头思故乡。";
          utterThis.lang="zh";
          utterThis.rate=0.9;
          speechSynthesis.speak(utterThis);
      }
    </script>
</head>
<body>
<input value="test" onclick="speak()" type="button">
</body>
</html>
```
