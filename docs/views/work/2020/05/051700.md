## 首先，找一张自己喜欢的图片，`png`格式

`manjaro`使用的主题位置`/usr/share/grub/themes/manjaro`

## 复制图片到指定位置

```bash
sudo cp background.png /usr/share/grub/themes/manjaro/background.png
```

## 更新配置

```bash
sudo grub-mkconfig -o /boot/grub/grub.cfg
```

## 重启系统

```bash
reboot
```
