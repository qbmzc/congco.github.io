## count.sh

```bash
# 查找两个文件的重复的数据输出到文件g
# 数据a即存在a.csv又存在b.csv中
grep -f a.csv b.csv > g.csv

# 对文件中的重复数据进行次数统计
sort g.csv | uniq -c > count.csv
```
