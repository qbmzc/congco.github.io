## 时间

```bash

```

## 计算分秒

```bash
$(($cost_time/60))

$(($cost_time%60))
```

## 写入文件

```bash
#!/bin/bash
 
start_time=$(date +%s)
sleep 1
 
#do something

end_time=$(date +%s)
cost_time=$[ $end_time-$start_time ]
echo "build kernel time is $(($cost_time/60))min $(($cost_time%60))s" > result.txt
```
