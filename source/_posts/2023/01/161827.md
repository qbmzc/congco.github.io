Mac M1安装程序后打开出现“已损坏，无法打开。 您应该将它移到废纸篓”

<!-- more -->



## 解决方案

```bash
sudo xattr -r -d com.apple.quarantine 
```

然后打开 Finder进入 “应用程序” 目录，找到该软件，将其图标拖到终端窗口，获取到文件路径，最终拼接为以下命令

```bash
sudo xattr -r -d com.apple.quarantine /Applications/XXX.app
```
